## Install

Ensure you have at least:

* node@7
* go-ipfs@0.4

Install the explorer:

```sh
npm install --global ipld-explorer-cli
```

Start your Go IPFS daemon:

```sh
ipfs daemon
```

## Usage

```sh
$ ipld-explorer

Welcome to the IPLD explorer REPL!
Type "help" then <Enter> for a list of commands

? >
```
